# n8n AI Automation Workflows

## Overview

This repository contains a collection of automation workflows developed using n8n. These workflows demonstrate the integration of AI models, email automation, webhooks, calendar management, and business process automation.

## Workflows

### 1. Email Classifier

* Automatically monitors incoming Gmail messages.
* Uses Google Gemini AI to classify emails.
* Applies labels based on email content.
* Reduces manual email management.

### 2. AI Calendar Assistant

* Integrates Google Gemini AI with Google Calendar.
* Checks calendar availability.
* Supports intelligent scheduling assistance.
* Uses external search capabilities.

### 3. Webhook AI Agent

* Receives requests through webhooks.
* Processes queries using Gemini AI.
* Returns automated responses.
* Suitable for chatbot and API integrations.

### 4. Delivery Routing System

* Routes requests based on workflow conditions.
* Sends notifications through email.
* Demonstrates business process automation.
